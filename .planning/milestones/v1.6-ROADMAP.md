# Milestone v1.6: Skills.sh Publication

**Status:** SHIPPED 2026-03-07

## Milestone Summary

**Key Decisions:**
- Plugin wraps, never modifies existing scripts (zero regressions, clean separation)
- Deterministic safety hooks via bash+jq (fast, free, predictable)
- Case-statement replaces declare -A for bash 3.2 macOS compatibility
- Scope auto-creation with localhost defaults instead of hard-deny
- Plugin skills are real file copies (not symlinks or hardlinks) for portability
- Marketplace descriptions match SKILL.md frontmatter exactly (single source of truth)
- Workflow dual-mode uses per-step branching (not per-section Mode headers)
- Agent frontmatter unchanged -- skills: field resolves by name: field matching
- Validation utility scripts excluded from BATS tool-contract test discovery

**Issues Resolved:**
- Bash 3.2 compatibility (replaced declare -A with case statements)
- Plugin symlinks replaced with real copies for portability
- validate-plugin-boundary.sh BATS failures (added metadata headers)
- PreToolUse hook false positives on git commit messages (temp file workaround)
- BATS associative array parsing (function-based resolution)
- macOS BSD sed incompatibility (awk-based section extraction)

**Issues Deferred:**
- None

**Technical Debt Incurred:**
- None -- all deferred items from earlier phases resolved during the milestone
